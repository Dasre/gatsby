## How to Use
```shell
# Clone the repository
$ git clone https://github.com/Dasre/gatsby

# Go into the repository
$ cd gatsby

# Remove older origin repository
$ git remote remove origin

# Install dependencies
$ npm install

# Start with gatsby
$ gatsby develop
```
